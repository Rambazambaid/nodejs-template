# NodeJS template

NodeJS project template can be used for frontend app as well as backend api app  

- typescript
- eslint

### Run project:

- "make install" (npm install)
- "make env" (create .env file)
- "make dev" (npm run dev)

###

- To see all avaliable Makefile commands use "make help"
